Practise 6: Attribute Directive & Pipe

About Practise:

Here we will see how to use attribute Directives for event handling and pipes for filtering.

Practise Details:

Starting from Directive

  • Create Directive (ng generate):
  • ng g directive [name]
ng generate directive [name]
  • The idea of this directive is to handle mouse events enter, leave and click
  • For this handling we will use @HostListener
  • @HostListener('mouseenter') onMouseEnter() {
    /** handling event **/
}
  • This directive should apply as attribute on all ChildBox components
    • On hover (mouseenter) it should apply background
    • Also when hover is happening, it should notify parent on ChildBox type (why?: this will be explained later )
    • Click event should also notify parent (why?: this will be explained later )

Our next step new Pipe

  • Create Pipe (ng generate):
  • ng g pipe [name]
ng generate pipe [name]
  • The idea of this pipe is to dynamically filter ChildBox-es in combination with parent template *ngFor directive.
  • <div class="parent"
    <div class="child" 
        *ngFor="let childBox of childrenBoxes | sortByType : selectedType">
    </div>
</div>

Here you can see what is developed so far with Parent and Child component.

Remember that their UI (border style) is the field of IChildBox where we defined component type.
